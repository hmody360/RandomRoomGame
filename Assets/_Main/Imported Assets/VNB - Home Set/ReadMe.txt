DETAILS:

The aim for this package is to just set-up things quickly, by providing ready-to-use prefabs of everything. This includes:

- Materials and original color palettes (that are pretty easy to modify).
(if you're using URP, then you will need to convert these to URP)
- Prefabs ready to snap into scene, to be animated or manipulated from script (some are actual ready-to-use compositions of models).

*Make sure to snap things to the grid when using tiles*

*Look at my itch.io page for details*

.-------------------.



--- Description ---

Simple low poly home-themed furniture set, containing all kind of stuff to populate a house, also including building tiles (wall, floor, and roof pieces, as well as doors, door handles, windows, etc..).

If you find bugs or any other kind of problem, please contact me on my itch.io page or through discord. I'm always open to any kind of suggestions on what to add or change!
