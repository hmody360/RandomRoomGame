# Setup Instructions:

## To Play the current build of the game:

- Go to https://hmody360.itch.io/rooms
- Install the zip folder below.
- Extract the Files from the ZIP Folder.
- Click the Applicaiton called Rooms (Application .exe File).
- A Windows Defender Pop up may appear, as this is marked as an executable foregin file, click view more then continue / or just continue.
- Enjoy :)
</br></br>

## To Check the Unity Project:
- Click on The Green Code Button Above.
- Copy the web URL within the pop up window.
- Within your the file where you want to save the project in, open Git Bash using Right Click.
- Type: git clone (Enter Copied Repository URL Here)
- After the Project is cloned to your device locally you need to import it in Unity.
- Open Unity Hub.
- Click Projects from the Left Menu.
- On the Top Right Click the Add Dropdown button.
- Click Add Project from Disk.
- Select the folder you cloned the Repo into.
- Voila you can click the project in Unity Hub to open it.
